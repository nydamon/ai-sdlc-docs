    Onboarding Phases¶

    1. Environment Setup (Day 1) - Tools and configurations
    2. Process Training (Day 1-2) - Workflows and best practices
    3. Validation (Day 2) - First contribution and review

    Hardware & OS Setup¶

    • [ ] System Requirements Check
    • [ ] macOS/Linux/Windows with WSL2
    • [ ] Node.js v16+ installed
    • [ ] PHP 8.1+ installed (for Laravel)
    • [ ] Python 3.8+ installed

    • [ ] Git latest version

    • [ ] IDE Installation

    • [ ] Install Cursor IDE
    • [ ] Install recommended extensions:
      • [ ] ESLint
      • [ ] Prettier
      • [ ] GitLens
      • [ ] Bracket Pair Colorizer
      • [ ] AI Code Review
    • [ ] Configure IDE settings from team template

    Tool Installation & Configuration¶

    • [ ] Core Development Tools
    • [ ] Run enhanced setup script: 
      curl -O https://raw.githubusercontent.com/nydamon/ai-sdlc-docs/main/enhanced-setup-script.txt
chmod +x enhanced-setup-script.txt
./enhanced-setup-script.txt

    • [ ] Verify installations: 

      node --version
npm --version
composer --version
cursor --version

    • [ ] Git Configuration

    • [ ] Set up Git user information: 
      git config --global user.name "Full Name"
git config --global user.email "email@company.com"

    • [ ] Configure Git aliases: 

      git config --global alias.st status
git config --global alias.co checkout
git config --global alias.br branch

    • [ ] AI Tool Setup

    • [ ] Configure Cursor API key: 
      export CURSOR_API_KEY="your-api-key"
echo 'export CURSOR_API_KEY="your-api-key"' >> ~/.zshrc
    • [ ] Install CodiumAI: 
      npm install -g codium
    • [ ] Verify AI tools: 
      cursor --version
codium --version

    Repository Access & Configuration¶

    • [ ] Clone Main Repository 

      git clone git@github.com:company/project.git
cd project

    • [ ] Install Project Dependencies 

      npm install
composer install

    • [ ] Configure Environment Variables

    • [ ] Copy .env.example to .env
    • [ ] Set required API keys and configurations

    • [ ] Verify database connections

    • [ ] Test Git Hooks 

      # Make a small change
echo "// test" >> test.js
git add test.js
git commit -m "test: onboarding setup"
# Should see automated formatting and checks
